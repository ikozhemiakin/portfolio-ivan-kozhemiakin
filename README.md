# Portfolio — Ivan Kozhemiakin

A modern, responsive portfolio website showcasing **Ivan Kozhemiakin** — Magento / Laravel Backend Developer with 8+ years of commercial eCommerce experience.

## Live demo

Open `index.html` locally or deploy to GitHub Pages / Vercel.

## Features

- **Modern dark UI** with glass morphism and gradient backgrounds
- **Fully responsive** — mobile, tablet, desktop
- **Sections**: Hero, About, Experience, Projects, Skills, Contact
- **CV downloads**: Laravel & Magento PDF resumes in `/public`
- **Links**: GitHub, LinkedIn, task-management repo

## Tech stack

- HTML5, CSS3, vanilla JavaScript
- No build step required — static deploy

## Project structure

```
portfolio-ivan-kozhemiakin/
├── index.html

└── README.md
```

## Local preview

```bash
cd portfolio-ivan-kozhemiakin
python3 -m http.server 8080
# open http://localhost:8080
```

## Deploy to GitHub Pages

1. Push repo to GitHub
2. Settings → Pages → Source: **Deploy from branch** → `main` / `/ (root)`
3. Your site: `https://ikozhemiakin.github.io/portfolio-ivan-kozhemiakin/`
